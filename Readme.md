## ğŸ“– Introduction

CV_TempleRun demonstrates how computer vision and gesture recognition can be used for hands-free interaction with games. Powered by [OpenCV](https://opencv.org/), [MediaPipe](https://mediapipe.dev/), and [PyAutoGUI](https://pyautogui.readthedocs.io/), this project captures hand movements via your webcam and translates them into keyboard commands to play Temple Run or similar endless runner games.  
This is a fun way to explore gesture recognition, automation, and computer vision in Python!
